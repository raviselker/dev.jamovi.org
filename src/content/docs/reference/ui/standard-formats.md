The following are the standard jamovi formats for use with the format property of UI controls:

| Format | Description |
| ------ | ----------- |
| `variable` | Formats the value as a variable name, identifying it with a variable icon. |
| `term` | Formats the value as an interaction term (e.g., A * B). |
| `number` | A standard numeric format. |
| `bool` | A standard boolean format (True/False). |
| `string` | A standard string format. |
